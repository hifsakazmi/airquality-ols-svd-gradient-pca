## Project Description

[Link to Dataset](https://www.kaggle.com/datasets/fedesoriano/air-quality-data-set)

### Libraries used:

* ucimlrepo (to import dataset directly via this library)
* kagglehub (to import dataset via kaggle link)
* pandas (data preprocessing)
* numpy (data processing and mathematical calculations)
* matplotlib (visualizations)
* seaborn (visualizations)
* sklearn (preprocessing, visualizations, train/test split, mean_squared_error)

### Methodologies Used
* OLS (Ordinary Least Squares)
* SVD (Singular Value Decompostion)
* Gradient-based optimization methods (Batch GD, SGD, Adam)
* PCA (Principle Component Analysis using SVD)
